## Açıklama

Projenin profesyonel standartlarda, ölçeklenebilir bir klasör yapısına sahip olması gerekiyor. Bu yapı, ileride ekibe katılacak geliştiricilerin kodu kolayca anlamasını sağlayacak.

---

## Kabul Kriterleri

- [ ] `src/components/` - UI bileşenleri için klasör
- [ ] `src/hooks/` - Custom React hooks için klasör
- [ ] `src/types/` - TypeScript type/interface tanımları için klasör
- [ ] `src/services/` - API çağrıları ve backend iletişimi için klasör
- [ ] `src/utils/` - Yardımcı fonksiyonlar için klasör
- [ ] `src/context/` - React Context (global state) için klasör
- [ ] `src/styles/` - Global CSS/stil dosyaları için klasör
- [ ] Her klasörde bir `index.ts` barrel export dosyası

---

## Teknik Notlar

**Barrel Export Nedir?**
```typescript
// components/index.ts
export { Calendar } from './Calendar';
export { EventCard } from './EventCard';
```

Bu sayede import'lar temiz olur:
```typescript
// Kötü
import { Calendar } from './components/Calendar/Calendar';

// İyi
import { Calendar } from './components';
```
