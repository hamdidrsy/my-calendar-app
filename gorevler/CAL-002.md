## Açıklama

Uygulamanın temel veri yapılarını TypeScript interface'leri olarak tanımla. Bu interface'ler tüm uygulama boyunca kullanılacak ve type safety sağlayacak.

---

## Kabul Kriterleri

- [ ] `Event` interface - Takvim etkinliği
- [ ] `CalendarDay` interface - Takvim günü
- [ ] `User` interface - Kullanıcı bilgisi
- [ ] `EventCategory` type - Etkinlik kategorileri (enum veya union type)
- [ ] `ViewMode` type - Takvim görünüm modları (günlük, haftalık, aylık)

---

## Beklenen Interface Yapısı

```typescript
// types/event.ts
interface Event {
  id: string;
  title: string;
  description?: string;
  startDate: Date;
  endDate: Date;
  category: EventCategory;
  createdBy: string;
  // ... düşün, başka ne lazım?
}
```
